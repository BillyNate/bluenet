33 /* Power Switch
34  *
35  * The Switch class governs the two (separate) pathways that enable power to the load (switch ON)
36  * or disable power to the load (switch OFF)
37  * The two pathways are:
38  * * the Relay (for high current loads but slow switching times)
39  * * the IGBTs (for low current loads but fast switching times)
40  * Switching the IGBT pathway is done through Pulse Width Modulation (PWM)
41  *
42  * #define variables used in this class
43  * - PWM_DISABLE If this is defined, the Switch class will ignore all attempts to switch PWM
44  * - CONFIG_PWM_PERIOD Time between consecutive switch ON actions during PWM dim cycle
45  * - CONFIG_RELAY_HIGH_DURATION Duration of time in which power is supplied to the relay in order to switch it
46  *
47  * */
