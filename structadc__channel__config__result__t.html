Detailed Description

Result struct after configuring an ADC channel.

Has all the info to put the sample values in context.

Usage: Vpin = sampleVal / maxSampleValue * maxValueMilliVolt * 1000 + Vpinref

pin: The AIN pin of this channel. referencePin: The AIN pin that was subtracted from the measured voltage on <pin>. samplingIntervalUs: The sampling interval in Î¼s. minValueMilliVolt: Minimum of the measured voltage (Vpin - Vpinref) range, in mV. maxValueMilliVolt: Maximum of the measured voltage (Vpin - Vpinref) range, in mV. minSampleValue: Minimum of the sample value range. maxSampleValue: Maximum of the sample value range.
