Detailed Description

template<class T, int U, class V>
class CyclicStorage< T, U, V >

Cyclic storage is a wrapper around flash memory.

The buffer stores individual elements together with a sequence number V to reduce wear on flash memory.

template<class T, int U, class V>
void CyclicStorage< T, U, V >::loadFromStorage ( )
inline

Load all values from memory.

A value can be stored at different locations in FLASH. The size of the buffer is given by the template parameter U. The loadFromStorage() function iterates through the values and sets the one with the largest sequence number and the sequence number itself in fields of the class.

The load function is automatically called from the constructor. After a store() it is not necessary to call loadFromStorage() before calling read().
