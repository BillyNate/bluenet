Detailed Description

template<typename T, int U = DEFAULT_PAYLOAD_LENGTH>
class StreamBuffer< T, U >

General StreamBuffer with type, op code, length, and payload.

General class that can be used to send arrays of values over Bluetooth. For the structure, see the <stream_t> struct.

typename T defines the type of the payload elements typename U defines the number of elements in the payload. by default, the number of elements is defined as (<MASTER_BUFFER_SIZE> - <SB_HEADER_SIZE>) / sizeof(T) but it can be overwritten if a smaller payload array should be used.

Sets pointer to zero, does not deallocate memory. Create a string from payload.

This creates a C++ string from the uint8_t payload array. Note that this not always makes sense! It will use the length field to establish the length of the array to be used. So, the array does not have to have a null-terminated byte.
