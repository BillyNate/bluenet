Detailed Description

template<typename T>
class StreamBuffer< T >

General StreamBuffer with type, length, and payload.

General class that can be used to send arrays of values over Bluetooth, of which the first byte is a type or identifier byte, the second one the length of the payload (so, minus identifier and length byte itself) and the next bytes are the payload itself.
