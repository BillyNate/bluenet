Detailed Description

ScheduleService organizes ticks for all components for non-urgent timing.

Constructor & Destructor Documentation

ScheduleService::ScheduleService ( )

Constructor for alert notification service object.

Creates persistent storage (FLASH) object which is used internally to store current limit. It also initializes all characteristics.

void ScheduleService::tick ( )
virtual

Perform non urgent functionality every main loop.

Every component has a "tick" function which is for non-urgent things. Urgent matters have to be resolved immediately in interrupt service handlers.
