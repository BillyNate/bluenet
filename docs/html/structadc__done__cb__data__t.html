Detailed Description

Struct storing data for ADC callback.

The struct adc_done_cb_data_t stores (1) a callback pointer, see adc_done_cb_t, (2) the arguments for this pointer, see also adc_done_cb_t (a pointer to the buffer with data from the ADC conversion, the buffer size, and a unique number for the buffer).
